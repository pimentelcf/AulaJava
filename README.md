# Aula Java 🚀
> Aqui são varios projetos feitos na faculdade e deixando tudo junto para revisar codigos e projetos realizados em sala de aula.

Este projeto foi desenvolvido com foco em **Java Moderno** e boas práticas de engenharia de software, aplicando conceitos de testes automatizados e gestão de dependências com Maven.

## 🛠 Tecnologias e Ferramentas
* **Java 21+
* **Maven

## 🏗 Estrutura do Projeto
O projeto segue o padrão de estrutura do Maven:
* `src/main/java`: Código-fonte da aplicação.
* `pom.xml`: Configurações de build e dependências.
